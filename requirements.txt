# engine skips every Polymarket setup ("no book data, skip") and never enters
# or notifies — which is exactly why only Limitless has been firing.
# Pinned to a known-good version that matches the import paths the code uses
# (py_clob_client.client.ClobClient, py_clob_client.clob_types.ApiCreds).
# NOTE: this pulls a heavy crypto dependency tree (eth-account, eth-abi, the
# web3 signing stack, pydantic 2.x), so the Railway build will be noticeably

# Limitless Exchange SDK — required for LIVE trading on Limitless (FOK orders,

# "Go Live" or "Approve USDC" records ERROR rows with "py-limitless missing").
# Bundles eth-account, web3, aiohttp, and python-socketio as transitive deps,
# which overlaps with what py-clob-client already pulled in — pip resolves the

py-limitless==0.1.1
